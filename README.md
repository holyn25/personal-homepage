# Personal Homepage

A React, Vite, and TypeScript personal resume homepage with generated cyber technology visuals.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Content

Replace the placeholder resume content in `src/data/profile.ts`. The page components read from that single data file, so names, links, experience, projects, skills, and contact text can be updated without changing the layout.

## Deployment

This is a static Vite site and can be deployed to Vercel or Netlify with the default settings:

- Build command: `npm run build`
- Output directory: `dist`
